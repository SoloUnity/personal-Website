yapOS

Description

yapOS is an OS simulator project in which I built to learn about the shell interface and basic OS functionalities such as process management, memory management, and file system operations. The design mimics a simplified OS kernel layered over a block device simulator. Concepts I learned in core systems programming include:

  • Variable storage and key-value substitution
  • Round-robin process scheduling with paging
  • Demand-paged memory management with LRU eviction
  • A persistent, FAT-style virtual file system
  • Script execution, command parsing, and a shell REPL
  • C
