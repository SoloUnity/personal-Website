Mars Rover Simulator

Description

The Mars Rover Simulator project simulates rover missions using real terrain data and advanced pathfinding algorithms. It is was a project to learn the core fundamentals of graphics, pathfinding algorithms, as well as group collaboration. This project involved a total of 7 students with work spanning several months. It includes features such as:

  • Custom built UI library with a frame by frame rendering of the UI
  • Create, save, load, and delete multiple projects, each with its own set of rovers and mission parameters
  • Interactive map with selectable areas of interest, path markers, and computed paths
  • Analyze path and rover statistics, including distance traveled, energy consumption, and mission duration
  • Projects and rover configurations are stored in a local database
  • Python
